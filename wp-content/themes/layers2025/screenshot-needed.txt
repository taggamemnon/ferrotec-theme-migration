Screenshot Required
==================

WordPress themes require a screenshot.png file for the theme appearance admin page.

Specifications:
- Filename: screenshot.png
- Dimensions: 1200 x 900 pixels (4:3 ratio)
- Format: PNG
- Shows: Homepage or representative theme layout
